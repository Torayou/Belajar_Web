# Rencana Pembelajaran Web Development

## Fokus Sementara:

### 1. **HTML (HyperText Markup Language)**

**Target Pembelajaran:**
- **Membuat Struktur Website yang Tepat**  
  Pelajari cara membuat halaman web dengan struktur yang benar menggunakan HTML. Fokus pada tag dasar, atribut, dan elemen-elemen penting dalam HTML untuk menyusun konten yang mudah diakses dan sesuai standar.
- **Pemahaman SEO Dasar**  
  Memahami bagaimana struktur HTML yang benar akan membantu dalam optimasi mesin pencari (SEO). Hal ini sangat penting agar halaman web dapat lebih mudah ditemukan oleh pengguna melalui pencarian di Google.

---

### 2. **CSS (Cascading Style Sheets)**

**Target Pembelajaran:**
- **Membuat Tampilan Menarik dengan CSS**  
  Pelajari cara menggunakan CSS untuk mendesain tampilan halaman web. Mulai dari pengaturan warna, font, margin, padding, hingga layout halaman seperti Flexbox dan Grid.
- **Dasar Responsive Design**  
  Fokus pada pembuatan layout yang responsif agar situs web bisa tampil dengan baik di berbagai perangkat (desktop, tablet, dan mobile).

---

## Lanjutan Pembelajaran:

### 1. **JavaScript**

**Target Pembelajaran:**
- **Membuat Logika Interaktif di Sisi Pengguna**  
  Pelajari cara membuat halaman web lebih interaktif dengan JavaScript. Misalnya, menambahkan efek interaktif seperti tombol yang bisa diklik, form validasi, dan animasi dinamis.
- **Integrasi dengan CSS**  
  Pelajari cara memadukan JavaScript dengan CSS untuk menciptakan pengalaman pengguna yang lebih dinamis dan menarik. Contohnya, mengubah elemen halaman secara langsung berdasarkan interaksi pengguna.

---

### 2. **PHP (Hypertext Preprocessor)**

**Target Pembelajaran:**
- **Membangun Logika Bisnis Sederhana di Backend**  
  Pelajari cara menggunakan PHP untuk menangani logika di server-side, seperti menerima dan memproses data dari form, interaksi dengan database, dan pengelolaan sesi pengguna.
- **Routing dan Middleware**  
  Memahami cara membuat routing sederhana, menangani permintaan HTTP, dan menggunakan middleware untuk memfilter atau mengelola alur aplikasi.

## Langkah-Langkah Pembelajaran:

1. **Mulai dengan HTML & CSS:**
   - Pahami struktur dasar HTML seperti `<html>`, `<head>`, `<body>`, dan elemen-elemen penting seperti `<div>`, `<header>`, `<footer>`, dan `<section>`.
   - Pelajari pengaturan layout dengan CSS, termasuk properti seperti `display`, `position`, `margin`, dan `padding`.
   - Fokus pada pembuatan desain yang responsif dengan menggunakan media query agar website dapat menyesuaikan diri dengan ukuran layar yang berbeda.
   
2. **Lanjutkan dengan JavaScript:**
   - Belajar cara menambahkan interaktivitas pada halaman, seperti menampilkan atau menyembunyikan elemen, merespons klik tombol, dan memvalidasi form.
   - Kembangkan pemahaman tentang DOM (Document Object Model) dan bagaimana JavaScript dapat digunakan untuk memanipulasi elemen HTML di halaman web.
   - Pelajari cara bekerja dengan API dan data dinamis untuk meningkatkan kemampuan interaktivitas aplikasi web.

3. **Penguasaan PHP untuk Backend:**
   - Pelajari dasar-dasar PHP untuk menangani form input, memproses data, dan menyimpan data ke dalam database (misalnya MySQL).
   - Pahami bagaimana membuat aplikasi backend sederhana dengan routing, middleware, dan pengelolaan sesi untuk autentikasi pengguna.
