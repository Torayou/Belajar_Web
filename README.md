### untuk belajar web developing

fokus sementara:
    1. HTML
    2. CSS

target:
    1. Mampu membuat web dengan struktur yang tepat (HTML), nanti nya bisa diperdalam untuk SEO.
    2. Mampu membuat tampilan dengan dasar struktur HTML dengan baik, nantinya bisa diperdalam untuk dipadukan dengan javascript untuk tampilan interaktif-dinamis (CSS).

lanjutan:
    1. javascript
    2. PHP

target: 
    1. Mampu membuat logika user side sederhana (interaktive), lanjutan dipadukan dengan css untuk tampilan lebih dinamis.(javascript)
    2. Memahami dan mampu membuat bussiness logic, middleware, routing untuk backend sederhana. Lanjutan bisa diperdalam untuk scaling dan securty.(node/php)
